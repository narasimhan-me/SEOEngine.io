# ENGINEO.AI — Brand Guidelines (v1.0)

## 1. Brand Essence

EngineO.ai is the first Discovery Engine Optimization platform — a system that optimizes your visibility across all AI and search engines:
SEO + AEO + PEO + VEO.

Brand personality:

- Intelligent
- Futuristic
- Precise
- Clean
- Trustworthy
- High-performance

Tone:

- Confident
- Clear
- Forward-thinking
- Expert but approachable

---

## 2. Visual Identity

### Primary Color

- **Electric Indigo — #516BFF**
  Represents intelligence, precision, and clarity across AI.

### Accent Color

- **Bright Cyan — #4FD1FF**
  Represents AI signals, discovery, and multi-engine optimization.

### Neutral Palette

- Slate Gray — #1F2937
- Deep Night — #111827
- Soft White — #F9FAFB

### Light/Dark Modes

- Equal priority for both.
- Use Electric Indigo as the leading highlight on both.

---

## 3. Logo System

### Logo Concept

“**O-shaped Engine Core**”

- Circular core representing an engine.
- Radial arcs representing AI discovery signals.
- Optional subtle "E" negative space inside the O.

### Logo Variants

1. Primary EngineO Core
2. Primary EngineO Wordmark
3. Small Icon (for navbar)
4. Dark mode variant
5. Light mode variant
6. Monochrome version

### Logo Geometry

- Circular base grid
- 3–4 radial arcs
- Arc thickness: consistent stroke
- Spacing ratio: 1.618 golden ratio style

---

## 4. Typography

### Primary Font

- **Manrope** (Headings / Logos)

### Secondary Font

- **Inter** (UI / Body text)

Usage:

- H1–H3: Manrope semi-bold
- UI components: Inter medium
- Paragraphs: Inter regular

---

## 5. UI Themes

### Light Mode

- Background: #F9FAFB
- Card: #FFFFFF
- Text: #111827
- Accent: Electric Indigo (#516BFF)

### Dark Mode

- Background: #111827
- Card: #1F2937
- Text: #F9FAFB
- Accent: Electric Indigo (#516BFF)

---

## 6. Iconography

- Round, geometric
- Subtle gradients allowed
- No sharp neon effects
- Use dual-tone indigo + cyan

---

## 7. Brand Voice

- Clear, confident, expert
- No hype
- Focus on transformation + intelligence
- Avoid SEO-only framing
- Always refer to “Discovery Engine Optimization (DEO)”
